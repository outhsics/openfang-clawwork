# Financial Reporting Case

This is a higher-value case-study folder designed to look closer to a serious internal finance delivery.

## Source task

- Occupation: `Financial Managers`
- Sector: `Finance and Insurance`
- Estimated task value: `$650.70`
- Source idea:
  Build a standardized Excel reporting package from a raw-data workbook, including branch and aggregate views, rankings, regional comparisons, and monthly/yearly operational metrics.

## Files in this folder

- [task_stub.json](task_stub.json): task metadata for OpenFang routing
- [workflow_run_prompt.txt](workflow_run_prompt.txt): prompt text for a workflow run
- [delivery/executive_summary.md](delivery/executive_summary.md): management-facing summary
- [delivery/branch_profitability_snapshot.csv](delivery/branch_profitability_snapshot.csv): sample structured output
- [delivery/metric_dictionary.md](delivery/metric_dictionary.md): output schema notes

## Why this case matters

This is closer to a real service product because it combines:

- structured analysis
- executive communication
- spreadsheet-style outputs
- repeated operational value

If the cleanup memo case proves "the system can deliver documents", this case proves "the system can support higher-value business reporting work."
